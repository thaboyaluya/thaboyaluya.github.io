Dark Mode

Astral framework also supports dark mode

Dark mode it's pretty straight forward to use. The automatic toggling uses the media-prefers-scheme to get the user's desired theme and apply automatically. Of course, this doesn't work in some (mosts) of Linux Operating systems, and that's what the manual version is aimed for. It's also really useful if you want your users to select their own theme.
